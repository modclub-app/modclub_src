# Modclub

Modclub is a moderation platform for online apps deployed on the IC. Moderation as a service (MaaS) is the concept of outsourcing controversial decisions or risky content in order to protect users and maintain trust within communities.

This repository will be updated shortly with the remaining wallet functionality that is currently under review. We have decided to share the code early so that the community can review the majority of the functionality. 

Once finalized we will be moving the code to our main repo https://github.com/modclub-app/modclub which will include all of our SNS configuration as well.
